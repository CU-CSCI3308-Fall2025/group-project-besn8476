# CU Marketplace

## Project Description

CU-Marketplace is a full-stack web application built for the University of Colorado Boulder community.
The platform allows users to:

- Create accounts and log in securely-
- Post items for sale, including price, category, images, and descriptions

- Browse listings with categories and sorting

- Edit or delete their own posts

- Search for items by keywords

- View item details and seller information

- The project is built using:

- Node.js / Express (backend API)

- Express-Handlebars

- PostgreSQL (database)

- pg-promise (database interface)

- Docker

- Mocha + Chai (testing)
